# ActiveBoost

Senior year team project at NYIT (2024). Built with a team of 5 — a fitness tracking and social challenge app with Fitbit API integration.

## What it does
A backend system for a fitness app where users can join groups, create challenges, track activity via Fitbit, earn points, and compete on a leaderboard.

## Features
- **Fitbit API integration** — pulls real activity data (steps, calories, active minutes) per user
- **Group challenges** — users create or join fitness challenges with defined goals and durations
- **Leaderboard** — points calculated from activity data, ranked within each group
- **User profiles** — tracks personal stats and challenge history

## Database design
Designed a relational schema with the following entities:
- Users
- Groups
- Group Memberships
- Challenges
- Participation Records
- User Profiles

## Tech stack
- **Python** — backend logic
- **Sanic** — async web framework
- **MySQL** — relational database
- **Fitbit API** — activity data integration
- **REST APIs** — client-server communication
